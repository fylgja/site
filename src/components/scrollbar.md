## Installation

```bash
npm install @fylgja/scrollbar
```

And include the component in to your code via;

```scss
@use "@fylgja/scrollbar";
// Or via PostCSS and other options as plain CSS
@import "@fylgja/scrollbar/scrollbar.css";
```
