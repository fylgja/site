### Display

| class        | value                    |
| ------------ | ------------------------ |
| block        | `display: block;`        |
| hidden       | `display: none;`         |
| inline-block | `display: inline-block;` |
| flex         | `display: flex;`         |
| inline-flex  | `display: inline-flex;`  |
| flow-root    | `display: flow-root;`    |

> Grid is not by default part of this group.
> And should be part of it own utility or component.
