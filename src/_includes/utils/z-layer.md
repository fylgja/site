### Z Index

| class  | value            |
| ------ | ---------------- |
| z-auto | `z-index: auto;` |
| z-0    | `z-index: 0;`    |
| z-1    | `z-index: 1;`    |
| z-2    | `z-index: 2;`    |
| z-3    | `z-index: 3;`    |
| z-4    | `z-index: 4;`    |
| z-5    | `z-index: 5;`    |

In case you need the index dept value to be bigger, like 10, 20 and 30.
Simply change the `$utilpack-index-dept` variable from 1 to 10.
