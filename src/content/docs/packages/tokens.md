Common defaults to make your body and typography look good
and easy to change to your needs.

## Installation

```bash
npm install @fylgja/base
```

Then include the component in to your code via;

```css
@import "@fylgja/base" layer(base);
```

```scss
@use "@fylgja/base";
```
